Abstractions

Intro

SySCoRe includes two possible abstraction methods,namely model-order reduction for continuous-state LTI systems and a finite-state abstraction for continuous-state systems. However, in order to create a finite-state abstraction of a nonlinear system we require an additional approximation step before constructing a piecewise-affine finite-state abstraction. Note that the piecewise affine approximation itself is considered a part of the finite-state abstraction method and not an abstraction method on its own.

Finite-state abstraction

To construct a finite-state abstraction of a continuous-state model, we grid the state space. More specifically, we compute the abstract state space as the set consisting of the centers of the grid cells. Next, the dynamics of the abstract model is defined by using the operator that maps states x to the center of the grid cell it is in. Details on how to construct a finite-state abstraction of a nonlinear system or an LTI system can be found in [1, Section III], and [2, Section IV] or [3, Section 4] respectively.
In SySCoRe, a finite-state abstraction is performed by the functions GridInputSpace and FSabstraction. The function GridInputSpace constructs the abstract input space by selecting a finite number of inputs from the input space. Next, we use FSabstraction to (among others) compute a probability matrix that contains the transition probabilities between states for all possible inputs in the abstract input space.

Piecewise-affine approximation

To approximate a nonlinear system by a piecewise-affin (PWA) system, we partition the state space and use a standard first-order taylor polynomial to approximate the nonlinear dynamics in each partition by affine dynamics. Additionally, we compute the error denoted by introduced by this approximation. In the toolbox, this is performed by the function PWAapproximation.
