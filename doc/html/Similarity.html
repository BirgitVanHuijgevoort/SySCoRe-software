The computation of the simulation relation relies heavily on the coupling of the inputs and disturbances of the two models. The inputs are coupled through an interface function and the disturbances via a coupling kernel. This is based on the method developed in [1].
More specifically, the underlying computation is based on finding an invariant set for the error dynamics . To this end, an optimization problem constrained by parameterized linear matrix inequalities is used to find a value for δ that corresponds with the given value of ϵ [1]. To solve this optimization problem we use the multi-parametric toolbox (MPT3) [3] with YALMIP [4] and with either solver SeDuMi [5] or MOSEK [6].
Similarity quantification is implemented in the function QuantifySim and the simulation relation itself is stored in the object SimRel.
